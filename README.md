# PPL.cz QA Testing Project

This repository contains a manual and automated QA testing project for the PPL.cz website, created as part of a diploma project.

## Project Overview
The goal of this project was to test the core user functionality of the PPL.cz delivery service website:
- Shipment tracking
- Pickup points map
- Website navigation and content pages

## Scope of Testing
### Manual Testing
- Functional testing of shipment tracking
- Map and pickup point search
- Navigation, localization (CS/EN), accessibility, adaptive layout
- Negative and edge case scenarios

### Automated Testing
Automated tests were implemented using:
- **Python**
- **Selenium WebDriver**
- **Pytest**
- **Allure Reports**

Automated test cases:
- TC-TRK-001 – Valid shipment tracking number
- TC-MAP-001 – Pickup point search by address
- TC-NAV-001 – Main menu structure and navigation
